Proyecto de phishing para capturar la imagen de la cámara de la víctima mediante un enlace falso.

- Basado en PHP, HTML y Bash.
- Genera páginas de acceso falso y registra IP/usuario.
- Diseño minimalista para pruebas en entornos controlados.

Requisitos
- Linux, WSL o Termux.
- Conexión a Internet.
- Bash disponible.

Uso
1. En Windows con WSL:
   - Instala WSL y una distribución Linux.
   - Abre la terminal de WSL.
   - Navega a la carpeta del proyecto.
   - Ejecuta: `bash hack.sh`

2. En Linux:
   - Abre una terminal.
   - Navega a la carpeta del proyecto.
   - Ejecuta: `bash hack.sh`

3. En Termux:
   - Abre Termux.
   - Navega a la carpeta del proyecto.
   - Ejecuta: `bash hack.sh`

Instalación automática
- `hack.sh` detecta e instala las dependencias necesarias:
  - `php`
  - `curl`
  - `wget`
  - `unzip`


Advertencia
